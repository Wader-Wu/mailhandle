# Example Prompts

Use these prompts only for the Codex skill mode. Standalone shell startup commands are documented in `README.md`.

Recommended entrypoint:

- `$mailhandle start`

Accepted aliases:

- `$mailhandle run`
- `$mailhandle launch`

## Start Workspace

- `$mailhandle start`
- `$mailhandle run`
- `$mailhandle launch`
- Start mailhandle and open my mail workspace.
- Launch mailhandle so I can review mail in the browser.
- Start mailhandle and open the workspace with browser auto-open.

## Daily Review

- Start mailhandle and help me review today's mail.
- Launch the workspace and show me today's highest-priority threads first.
- Open mailhandle so I can check which threads from today already have my reply.
- Start the workspace and let me triage today's mail by status and priority.

## Yesterday Review

- Start mailhandle and help me review yesterday's mail.
- Launch the workspace and filter it to yesterday's important emails.
- Open mailhandle so I can check what follow-up is needed from yesterday.

## Project Mail

- Start mailhandle and help me review this month's `zhuque` mail.
- Open the workspace and filter it for Zhuque-related threads.
- Launch mailhandle so I can find high-priority project emails quickly.

## Sender-Based Review

- Start mailhandle and help me review recent mail from `Jeffrey`.
- Launch the workspace and focus on unread emails from `Supannee`.
- Open mailhandle so I can inspect recent threads from `agilep@lumentum.com`.

## Response Workflow

- Start mailhandle and help me draft a reply for an important thread.
- Launch the workspace and open the latest Outlook item in this thread.
- Open mailhandle so I can generate a reply draft and use Reply All in Outlook.

## Priority Tuning

- Start mailhandle so I can edit priority rules in the browser.
- Launch the workspace and open the embedded priority-rules editor.
- Open mailhandle and help me tune the rules after I review this week's mail.
