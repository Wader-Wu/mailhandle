# Example Prompts

Use these prompts directly in OpenClaw.

## Daily Review

- Read my unread emails from today and summarize them as prioritized todos.
- Check today's mail and tell me what I should do next.
- Review today's unread Outlook emails and group them into high, medium, and low priority.
- Review today's mail and tell me which items I already replied to.
- Review today's mail and separate items that still need action from ones I already replied to.

## Yesterday Review

- Read yesterday's emails and summarize action items for me.
- Show me yesterday's important emails and what follow-up is needed.
- Review yesterday's Outlook mail and turn it into a todo list.
- Check yesterday's mail and mark which threads already have my reply in Sent Items.

## Project Mail

- Find emails with subject containing `zhuque` from this month and summarize the next actions.
- Review the `zhuque` project emails from this month and summarize the next actions.
- Show me unread Zhuque-related emails and tell me which ones are highest priority.
- Review this month's Zhuque emails and create a concise worklist.

## Sender-Based Review

- Find emails from `Jeffrey` this month and summarize what I need to do.
- Show unread emails from `Supannee` and turn them into prioritized todos.
- Review recent mail from `agilep@lumentum.com` and tell me if anything needs action.

## Tight Filtering

- Read unread emails from today with subject containing `urgent` and summarize the action items.
- Check emails from the last 7 days with subject containing `review` and create a todo list.
- Show emails from yesterday from `Luke` and summarize follow-ups.

## With More Structure

- Read my unread emails from today. Return a prioritized todo list with title, sender, received time, and next action.
- Review this month's emails about `zhuque`. Summarize the main themes and list the concrete next steps.
- Find yesterday's unread emails and tell me which ones are likely only notifications versus real work items.
- Show me every matching email separately without collapsing threads.
- Include notification-style emails as well so I can review everything.
- Review this week's inbox and annotate any item I already responded to.
- Format the report as two sections: needs action and already replied.

## Priority Tuning Prompts

- Review my unread emails from today using the current priority rules and tell me which rule affected each result.
- Read today's emails and tell me whether the current priority rules need adjustment.
- Summarize this week's emails and point out any false positives in the current priority settings.
